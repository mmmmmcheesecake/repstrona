Import produktów

Wklej dane ze spreadsheetów Google Sheets, Excel lub CSV — produkty zostaną dodane do katalogu.

1
Skopiuj dane ze spreadsheetów
Jak to zrobić:
1. Otwórz swój Google Sheets / Excel
2. Zaznacz wszystkie komórki z produktami (włącznie z nagłówkami)
3. Skopiuj Ctrl+C
4. Wklej poniżej Ctrl+V

Oczekiwany format: kolumny oddzielone tabulatorami lub przecinkami.
Przykład kolumn: Nazwa Kategoria Cena Link Zdjęcie Opis
2
Dopasuj kolumny

3
Podgląd zaimportowanych produktów
4
Pobierz i wgraj na stronę
Po pobraniu pliku:
1. Zastąp nim plik products.json w folderze strona na pulpicie
2. W terminalu VS Code wpisz: .\push.bat
3. Produkty pojawią się na stronie automatycznie ✓
